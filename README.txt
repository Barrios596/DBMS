# Bases de Datos
# DBMS

Bienvenido al Manejador de Bases de Datos DBMS. La carpeta src contiene todos los archivos necesarios para poder utilizar DBMS. Es necesario primero compilar todos los archivos .java que se encuentren dentro de esta carpeta previo a su uso. El archivo Main.java es el que debe correrse para ejecutar el programa.

Al ejecutar, aparecerá un área de texto al lado izquierdo, en donde podrá ingresar un programa escrito en SQL para crear bases de datos, eliminarlas, crear tablas, eliminarlas, modificarlas, agregar entradas, agregar restricciones, entre otras operaciones que se permiten en este lenguaje. Si desea saber qué expresiones puede ingresar a DBMS, consulte la gramática GramaticaSQL.g4 que se encuentra en la carpeta src. Cualquier error de sintáxis se desplegará en una ventana nueva, mientras que los errores semánticos y los resultados de las instrucciones se despliegan en el área de texto al lado derecho.

Todos los programas creados por los desarrolladores se encuentran debidamente comentados y con un encabezado que describe su utilidad.
